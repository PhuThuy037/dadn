## ⚙️ Run project `.env`

```
Tạo file .env ở src folder
Copy toàn bộ file .env.example  qua .env

Tạo file .env ở frontend folder(nếu chưa có)
Copy toàn bộ file .env.example  qua .env


```

## Run docker

```
Chạy theo từng lệnh
1. docker-compose build
2. docker-compose up -d
3. docker-compose exec app bash 
4. composer install
5. php migrate 
 
```
